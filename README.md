# Global Recipe Insights 
## Project Objective
The purpose of this dashboard is to analyze recipes across different cuisines, categories, cooking methods, difficulty levels, and continents. It provides insights into cooking times, popularity, and ingredient usage. This can help food bloggers, restaurants, and chefs better understand culinary trends worldwide.

## Dataset Link 
- <a href="https://github.com/prachisonaje/excel-analytics-dashboard/blob/main/Food_Recipes.xlsx"> Dataset </a>

## Questions (KPIs)
- Which cuisine has the highest average rating?
- Which recipe takes the least time to prepare?
- What is the average cooking time across all cuisines?
- Which ingredients appear most frequently in top-rated recipes?
- Which cuisine has the maximum number of recipes?
- What is the distribution of recipes by difficulty level (Easy, Medium, Hard)?
- Which recipe has the highest user rating?
- How do ratings vary with preparation time?
- Which cuisine offers the healthiest (or lowest-calorie) recipes? (if you have nutrition data)
- What percentage of recipes are rated above 4 stars?

## Process
1.Data Collection
- Collected food recipe data including cuisine type, ingredients, cooking time, difficulty, and ratings.
- Stored data in an Excel file (Food_Recipes.xlsx).

2.Data Cleaning
- Removed duplicate and blank entries.
- Standardized cuisine names and difficulty levels.
- Formatted numeric and categorical data for consistency.

3.Data Preparation
- Created calculated columns (e.g., average rating, total time, or category counts).
- Organized data into structured tables suitable for pivot analysis.

4.Dashboard Development
- Built pivot tables to summarize key metrics (ratings, cuisines, ingredients, etc.).
- Added charts (bar, pie, and line) for visual insights.
- Used slicers for interactive filtering by cuisine, difficulty, and rating.

4.Visualization & Insights
- Designed an interactive dashboard layout.
- Highlighted key KPIs such as top-rated cuisine, average cooking time, and most popular ingredients.
- Drew conclusions and insights based on visual trends.
